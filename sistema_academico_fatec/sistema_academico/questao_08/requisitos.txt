============================================================
SISTEMA DE GESTÃO ACADÊMICA - FATEC-PB
Módulo: Emissão/Impressão do Comprovante de Inscrição
============================================================

REQUISITOS FUNCIONAIS
----------------------------------------
RF01 - Buscar inscrição por CPF ou nome do candidato
RF02 - Exibir ficha de inscrição completa na tela
RF03 - Gerar comprovante de inscrição em formato imprimível
RF04 - Opção de salvar o comprovante
RF05 - Opção de imprimir diretamente pelo sistema
RF06 - Validar se candidato possui inscrição ativa antes de emitir

REQUISITOS NÃO-FUNCIONAIS
----------------------------------------
RNF01 - Comprovante gerado somente após cadastro de inscrição
RNF02 - Acesso disponível para usuário e administrador
RNF03 - Layout do comprovante claro e legível
RNF04 - Banco de dados SQLite
RNF05 - Sistema disponível via navegador web
